## Overview

This repository contains a curated list of **52 watch brands** that are actively monitored or tracked by [WatchPolice.com](https://watchpolice.com).

The goal is to maintain a clean, up-to-date reference of:
- Microbrands
- Value / accessible watch brands
- Select established brands (primarily for limited edition tracking)

This list powers WatchPolice’s release monitoring, limited edition alerts, and research tools.

## The Brands List

📄 **[View the full brands list →](brands.md)**

The main data file is located at [`brands.md`](brands.md). It includes:

- Brand name
- Official website
- Type (Microbrand / Value / Established)
- Primary focus (Dive, Tool, Field, Dress, etc.)
- WatchPolice tracking status
- Notes

## Why This Exists

WatchPolice.com focuses on **microbrand watches** and limited editions. This repository serves as:

- A transparent reference for users
- A structured dataset for internal tools
- A starting point for anyone researching current watch brands

We prioritize quality over quantity and focus on brands that regularly release new models worth tracking.
